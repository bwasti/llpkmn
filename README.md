# How To

1. Download mGBA https://mgba.io/downloads.html
2. Load a Pokemon ROM and start the game
3. Open scripting and load the server.lua (from https://github.com/nikouu/mGBA-http)
   Tools > Scripting
   File > Load Script > server.lua
4. `python ai.py`
